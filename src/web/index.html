schemaSkill, verilen bir URL'yi Playwright ile tarar, sayfa içeriğini analiz eder ve Azure OpenAI kullanarak Schema.org JSON-LD yapılandırılmış verisi üretir. Üretilen şema doğrulanır ve Google'ın zengin sonuç gereksinimlerine göre değerlendirilir. Toplu denetim modu ile birden fazla sayfayı aynı anda analiz edebilir ve raporlar oluşturabilirsiniz.

npm install
cp .env.example .env
# .env dosyasını Azure OpenAI bilgilerinizle düzenleyin
npx playwright install chromium

CLI Kullanımı

Tekil Schema Üretimi

npx tsx bin/schemaSkill.ts generate <url> [--type <type>] [--output <file>]

# Örnekler:
npx tsx bin/schemaSkill.ts generate https://example.com/blog/post
npx tsx bin/schemaSkill.ts generate https://example.com/product --type Product --output schema.json

Toplu Denetim

npx tsx bin/schemaSkill.ts audit <input.csv> [--output-dir <dir>] [--concurrency <n>]

# Örnek:
npx tsx bin/schemaSkill.ts audit urls.csv --output-dir ./reports --concurrency 2

Desteklenen Schema Tipleri

Araç, schema.org vocabulary dosyasını dinamik olarak indirip 800+ tipi destekler. Özellikle optimize edilmiş tipler:

  • Article / NewsArticle / BlogPosting — Blog yazıları, haberler
  • Product — E-ticaret ürün sayfaları
  • FAQPage — Sıkça sorulan sorular
  • LocalBusiness / Restaurant — Yerel işletmeler
  • Organization — Şirket/kuruluş sayfaları
  • WebSite — Ana sayfa, site geneli
  • Event — Etkinlik sayfaları
  • Recipe — Yemek tarifleri
  • JobPosting — İş ilanları
  • Course — Eğitim/kurs sayfaları
  • HowTo — Nasıl yapılır kılavuzları
  • Review / AggregateRating — Değerlendirmeler
  • VideoObject — Video sayfaları
  • BreadcrumbList — Breadcrumb navigasyonu
  • SoftwareApplication — Yazılım ürünleri

Mimari

Proje modüler bir yapıda organize edilmiştir:

  • crawler/ — Playwright ile headless browser tarama, sayfa içerik çıkarma, mevcut schema tespiti
  • schema/ — Schema.org vocabulary yönetimi, sayfa tipi algılama, JSON-LD doğrulama
  • ai/ — Azure OpenAI entegrasyonu, prompt şablonları, schema üretim/denetim
  • reporter/ — Markdown ve CSV rapor üretimi
  • input/ — CSV okuma, URL doğrulama
  • utils/ — Yapılandırma, disk cache, rate limiter, ilerleme göstergesi

API Referansı

POST /api/generate

Tek bir URL için schema üretir.

// İstek
{ "url": "https://example.com", "type": "Article" }

// Yanıt
{
  "crawlResult": { "pageContent": {...}, "existingSchemas": [...] },
  "detectedType": { "type": "Article", "confidence": 0.85, "signals": [...] },
  "usedType": "Article",
  "vocabularyInfo": { "type": "Article", "properties": ["headline", "author", ...] },
  "generatedSchema": { "@context": "https://schema.org", "@type": "Article", ... },
  "validationResult": { "valid": true, "errors": [], "warnings": [] }
}

POST /api/audit

Birden fazla URL'yi denetler.

// İstek
{ "urls": ["https://example.com", "https://example.com/blog"] }

// Yanıt
{
  "results": [{ "url": "...", "auditResult": {...} }],
  "summary": { "totalUrls": 2, "crawled": 2, ... },
  "markdownReport": "# Schema Markup Denetim Raporu\n..."
}
